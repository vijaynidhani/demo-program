# hello-springboot

Simple Spring Boot app exposing GET /hello returning "Hello, world!"

Run with Maven (requires Maven installed):

```powershell
cd "c:\work\AI Reloaded\Resta\hello-springboot"
mvn spring-boot:run
```

Or build and run the jar:

```powershell
cd "c:\work\AI Reloaded\Resta\hello-springboot"
mvn package
java -jar target/hello-springboot-0.0.1-SNAPSHOT.jar
```

Endpoint: http://localhost:8080/hello
